# lailara-palette

Color and typography constants from the Lailara Design System v2.

Vendored into Decompose (Cinderhaven sales-penetration tool #3) so the app has a
single, versioned source of truth for brand tokens — identical to the copy used by
Spin Rate. Import tokens directly:

```python
from lailara_palette import LL_CANVAS, LL_CHICAGO, LL_SEQ, LL_STATUS
```

Canonical source: https://github.com/MsShawnP/lailara-design-system
